# FIRST
A nodeJS \ VueJS single page web app for posting and sharing revision materials on various courses. Backend written in nodeJS + Express. Frontend uses VueJS.

To install:
Download nodeJS and run 'npm install' in this directory. Create an RSA keypair and name them 'public.key' and 'private.key' in this directory.
Change directory into /client and run 'npm install' there then 'npm run build'. See readme there for more info.

Run node index.js and navigate to the site on localhost.

Site demo at https://protected-citadel-29742.herokuapp.com/#/

The user "doctorwhocomposer" can log in with any password and is an admin.
The user "jcktwd" can log in with the password "password" and is not an admin. Have fun.
